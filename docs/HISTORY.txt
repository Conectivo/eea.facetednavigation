3.0 (2010-06-16)
----------------

Upgrade steps
~~~~~~~~~~~~~
* Within portal_setup import all steps using 'EEA Faceted Navigation' profile
  or reinstall it from portal_quickinstaller.

Issues and enhancements
~~~~~~~~~~~~~~~~~~~~~~~
* #3409 faceted_view.css has no condition anymore in portal_css profile. Removed
  duplicate from faceted_edit.css [voineali]
* #3059 Made Large Folder faceted navigable [voineali]
* #3338 Added special faceted widget ETag in order to easily generate another
  faceted version id which is used by squid or
  other proxy caching systems. See docs/versions.txt [voineali]
* Cleanup old css/js profile by removing duplicated items (@@ ones) [voineali]
* #3275 Make faceted navigation aware of collection's query.
  [Thanks to Izhar Firdaus]
* #3275 Moved faceted actions from portal_fiveactions to portal_actions
  as as it caused conflicts with p4a.plonecalendar and p4a.video.
  [Thanks to Izhar Firdaus]
* Refactored field.Vocabulary to be reused also with other fields
  than StringField [voineali]
